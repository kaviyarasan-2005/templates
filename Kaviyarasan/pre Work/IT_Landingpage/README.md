# TechAssist - IT Support Website

## Project Structure
```
├── index.html          (Home V1)
├── index-v2.html       (Home V2 - Dark split layout)
├── contact.html        (Contact form + info)
├── services.html       (6 service cards)
├── help-center.html    (FAQ Accordion)
├── pricing.html        (3-tier pricing)
├── login.html          (Auth - Login)
├── register.html       (Auth - Register)
├── about.html          (About + Team)
├── admin-dashboard.html (Admin panel)
├── 404.html            (Error page)
├── coming-soon.html    (Launch countdown)
└── assets/
    ├── css/style.css   (All styles - Flexbox/Grid, RTL)
    ├── js/main.js      (Nav, accordion, pricing toggle)
    └── js/rtl-toggle.js (LTR/RTL direction switcher)
```

## Features
- Pure HTML5, CSS3, Vanilla JS (no frameworks)
- Mobile-first responsive (320px to 4K)
- RTL/LTR toggle via globe icon in header
- FontAwesome 6 icons (CDN)
- Inter font from Google Fonts
- Tri-color scheme: Deep Blue, Electric Cyan, Clean White
- Unsplash stock images (swap paths as needed)

## Getting Started
Open `index.html` in a browser or serve with any static file server.
