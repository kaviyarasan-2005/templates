# FORXA Manufacturing Website

## Project Structure

```
forxa-website/
├── index.html          (Home V1)
├── index-v2.html       (Home V2)
├── about.html          (About Us)
├── contact.html        (Contact)
├── services.html       (Services)
├── products.html       (Products)
├── infrastructure.html (Infrastructure/Facilities)
├── blog.html           (Blog/News)
├── pricing.html        (Pricing)

├── login.html          (Login)
├── register.html       (Register)

├── coming-soon.html    (Coming Soon)

│   └── icons/
```

## Features
- 15+ fully responsive HTML pages
- Pure CSS (Flexbox/Grid) — no frameworks
- RTL/LTR toggle with globe icon
- Mobile hamburger menu
- Accordion FAQ
- Admin dashboard with sidebar
- Stock images via Unsplash
- FontAwesome 6 icons
- Google Fonts (Inter)

## Setup
Just open index.html in a browser. No build step required.

## Color Palette
- Primary: #0A2647 (Dark Navy)
